## ðŸš€ Getting Started

### Prerequisites

-   Node.js (>= 14.0.0)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/RynnKunnn/Rynn-UI.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Rynn-UI
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Customize `settings.json` to your liking.
5.  Start the server:
    ```bash
    npm start
    ```

Your API documentation will be available at `http://localhost:<PORT>`.
